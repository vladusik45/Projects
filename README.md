
# Projects

Проекты разбиты по разным каталогам в зависимости от основного языка, на котором они были написаны. Основные языки - C# и Python. Также разрабатывались парочка веб-интерфейсов на JS и с библиотекой React. Во многих проектах использовалась локальная база данных. Некоторые проекты представляют собой курсовые работы, финальные лабораторные работы, а небольшая часть - обычные небольшие лабораторные работы, которые я также хотела добавить в данный репозиторий. 

Среди проектов есть полноценные веб-сайты, API, telegram-bot с ML, несколько скриптов лабораторных работ на Unity, проекты с docker, парсеры, тесты и т.д.


## Структура репозитория
Каждый католог отвечает за проекты, написанные на определенном языке программирования. К некоторым дам объяснения ниже.
 - [CSharpLabChecker](csharp/CSharpLabChecker/CDiazCodeLab) - ответвление для проектной работы по автоматической проверке лабораторных работ. Проект изначально мог проверять работы только на Python, отдельным заданием была разработка дополнительного сервиса по компиляции и проверке ответов кода на C#. Код, отправленный в тело запроса (текстом или в виде файла), компилируется, сравнивает ответы с заранее подготовленными ответами для этого набора данных, подсчитывает длину кода, а также размер файла, отправленного для проверки. Разрабатывался проект в паре.
 - [StudyProjects](https://github.com/vladusik45/Projects/tree/main/csharp/StudyProjects) представляет собой лабораторные работы на C#.
 - [HHProjectWebApp](https://github.com/vladusik45/Projects/tree/main/csharp/HHProjectWebApp) - проект наподобие известного сервиса HH. Внутри есть также frontend для API. Также в проекте использовалась локальная база данных.
 - [FastApi](https://github.com/vladusik45/Projects/tree/main/python/fastAPI) - API сервиса продажи недвижимости для нескольких лабораторных работ. Внутри также есть сервисы для работы с чужим API, файл docker-compose разворачивает сервер для обмена сообщениями через rabbitmq.
 - [Python tests](https://github.com/vladusik45/Projects/tree/main/python/python_tests) собирает в себе несколько лабораторных работ по тестированию.
 - [Study_projects](https://github.com/vladusik45/Projects/tree/main/python/study_projects) представляет собой лабораторные работы на Python.
 - [Utils_for_adminsgta5rp](https://github.com/vladusik45/Projects/tree/main/python/utils_for_adminsgta5rp) - мои полезные скрипты для упрощения администраторской деятельности на проекте GTA 5 RP. Помимо помощи игроками, проверки жалоб и ответов на репорты у администраторов были задачи в проверке всевозможных логов фракций или отдельных игроков, также нужно было отслеживать "слив склада" (присвоение большого количества имущества организации себе без разрешения лидера и администрации), что было самым сложным и долгим. Для ускорения монотонной работы выкручивалась как могла, начала писать разные скрипты по детальной проверке всех строчек логов. Перед этим все логи парсились с админского сайта самостоятельно. Последней моей работой было составление таблиц активностей фракций в Google Sheets - был создан аккаунт разработчика и подключена таблица, там по запуску скрипта парсились все логи, обрабатывались каждые строчки, подсчитывалась статистика по каждой отдельной организации и записывалась в таблицу. Этот проект считаю одним из своих лучших, потому что делался для себя, для автоматизации процессов, а не по нужде закрыть предметы по учебе.
